Requerimientos



1. Generalización de papeletas: Este generara un conteo de las papeletas que se usaran en el sistema.

2. Generalización de resultado: El TSE dará a conocer a los departamentos el resultado final por medio de una gráfica.

3. Controlador votos: Después de que los usuarios hayan emitido el voto Deberán poder controlar los datos que estos procesaran en el sistema.

4. Formularios por cada tabla de una base de datos: En este proceso podemos desarrollar todas las funciones del software, ya que con esto generamos de la base de datos y podemos saber cada nombres de campos, tipo de datos y longitud de los datos de cada tabla, lo cual permite el poder controlar las funciones que serán base para el funcionamiento del sistema.

5. Validación de entrada de datos de formularios: Los formularios creados automáticamente deberán poder controlar los datos que estos procesaran, en este caso se utilizará la validación básica que todos los campos deberán de contener datos.

6. Generación de modelos de datos: Los modelos de datos son las especificaciones de como asociar los datos de cada tabla de la base de datos en una forma de objeto, para así poder generar el encapsulamiento, herencia, poder trabajar sobre una metodología de programación orientada a objetos.

7. Generación de páginas con resultado: Estos componentes serán nuestra salida básica de información y así los usuarios sepan los movimientos e información con la que cuenta el sistema.
